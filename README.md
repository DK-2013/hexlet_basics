##
### Requirements

* Mac / Linux
* Docker
* Docker Compose
* Ansible (installed using pip3)

### Install

```sh
$ make compose-setup
```

### Load content

```sh
$ make app-exercises-load
```

### Run

```sh
$ make compose
```

Go to [http://localhost:4000/](http://localhost:4000/)

    brew install git-secrets
