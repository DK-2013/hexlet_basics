### Requirements

* Mac / Linux
* Docker
* Docker Compose

### Install

```sh
$ cd hexlet_basics
$ make compose-build
$ make compose-setup
```

### Load content

```sh
$ make compose-bash
$ mix x.exercises.load php
```

### Run

```sh
$ cd hexlet_basics
$ make compose
```
